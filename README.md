# NAD Discovery
Explore the Monad ecosystem in traditional grid view or immersive 3D space mode.
Features
Homepage

Search and filter projects by category
Interactive 3D particle background (particles with mouse interaction)
6 floating project logos in 3D space
Real-time project updates via Firebase

# Space Mode

Navigate through 3D space using WASD/arrows or mouse
Projects displayed as textured spheres in a spiral formation
Hover for project details, click to visit websites
5000-star background with smooth camera controls

# Admin Panel

Password-protected project management
Add/edit/delete projects with multi-category support
Review and approve community submissions
Search and filter functionality

# Project Submission

Public form for adding new projects
Multi-category selection
Pending/approved workflow

# Stack

Frontend: HTML5, CSS3, JavaScript ES6+
3D Graphics: Three.js r128
Database: Firebase Realtime Database
Typography: Google Fonts (Inter)
